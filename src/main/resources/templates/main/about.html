About

INITIAL PROPOSAL

The goal of my web application was to help people looking for a room to rent as well as those looking to advertise a room, as shared accommodations have become more popular in Ireland because of the housing crisis.

The Renter user of this application would be able to search for rooms by location (with advanced filtering options), schedule and manage viewings. While a Sharer would be able to list a room and manage its listings and viewings.

The Google Maps API will be used to get the coordinates of the listings and then used as input in the Walker Score API, which will display how walkable the area is.

MODIFICATIONS

I tried to implement the scheduling functionality into my application, but due to its complexity and on Professor Keith's recommendation, I decided to add a contact form to the listing details page instead so that Renter users could send inquiring messages instead.

When a listing receives more inquiries than a predetermined number, a warning is given in the inquiry form informing users that the Sharer is likely to disregard their contact.
