INITIAL PROPOSAL

The goal of my web application was to improve the renting process of shared accommodations in Ireland, which has grown in popularity as a result of the housing crisis, by addressing issues identified in similar housing applications in the Irish market, such as the lack of scheduling functionality for property viewings.

Two main users are able to access my application:

• Sharer user – allowed to create listings of shared accommodations by inputting its detailed information, such as address, price, type of room, number of house mates, and viewing time slots availability. When all the viewing timeslots have been filled, the listing will become immediately unavailable. In the Sharer Dashboard this user will be able to view, edit or delete their listings and manage the viewings that have been scheduled by renter users.

• Renter – allowed to browse the accommodations listed, being able to search by location and apply advanced filtering options. After selecting the preferred listing and viewing their details, they will be able to choose the timeslot preferred and submit their contact details to the sharer. In the Renter Dashboard, the user will be able to view, reschedule or cancel all their viewings.
